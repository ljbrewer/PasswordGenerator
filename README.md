# PasswordGenerator

The password can include special characters, Uppercase Letters, Lowercase Letters, and digits. 

The user is prompted to choose all of these or just some.

It randomly generate a password that meets certain criteria so that it creates a strong password that provides greater security.

Then the user can choose the length of the password from 8 to 128 characters.

The program uses the number of choices and the length of the password to determine how many passes thru the loop are needed to generate the password of the length requested.

When the button is click it dispays the password on the page for the user.

This Password Generator application displays a red button to "Generate Password".
